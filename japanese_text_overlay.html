ATRI - 日语文字叠加展示

日语句子信息

上面展示的是通过STORY目录下的图片叠加形成的完整日语句子。每张图片包含一个或多个日语字符，当它们按照顺序紧密排列时，就形成了一句完整的日语。

实现原理

核心实现方式

这个日语文字叠加效果通过以下方式实现：

1. 图片排列技术

.image-group { display: flex; justify-content: center; align-items: center; gap: 1px; /* 极小间距，让图片几乎紧贴 */ }

2. 图片样式设置

.japanese-image { max-height: 200px; object-fit: contain; transition: all 0.3s ease; }

3. 交互效果

每张图片都有悬停效果，当鼠标悬停在某个字符上时，该字符会轻微放大并提升阴影，增强用户体验。

4. 动态加载

使用JavaScript动态加载STORY目录下的所有图片，并按照文件名顺序排列，确保形成正确的日语句子。
